# WebComponents-Lit

Proyecto de aprendizaje de Lit y Web Components.

## Qué contiene

- `src/index.html`: página de ejemplo que carga el componente Lit.
- `src/app.js`: componente Lit básico (`MiSaludo`) con propiedades y estilo.
- `src/repaso_js/`: archivos de repaso de JavaScript sobre clases, herencia, instancias y módulos.

## Cómo usarlo

1. Instala dependencias:
   ```bash
   npm install
   ```

2. Abre `src/index.html` con un servidor local. Por ejemplo:
   ```bash
   npx http-server src
   ```

3. Navega a `http://localhost:8080` (o el puerto que use el servidor).

> El ejemplo Lit usa módulos ES, por lo que es mejor servir los archivos desde HTTP y no abrirlos directamente con `file://`.
